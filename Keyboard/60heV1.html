Welcome to My Wooting 60HE Build Breakdown

I wanted to try to answer some of the questions I had, before I started my build, just in case anyone wants to do something similar in the future. This is my first keyboard build, so I'm still learning new things!

Let's get started:

Sound Dampening:

Silicone Mold Kit | $10.99

Started with the Stock Wooting foam in the Aluminum GH60 case. It sounded okay, but there was still a lot of room left in the case, which caused some case ping. Personally had to change it out for something that fills more space in the GH60.

To start the silicone mold, I covered all holes in the GH60 case with masking tape, including standoffs. Stirred the silicone with a plastic fork, in a solo cup, for a little over 6 minutes. Left it to cure overnight for about 12 hours. I ended up going through with an exacto knife, to remove some extra material around the standoffs. So, it doesn't look as perfect as it was, when it finished curing but it sounds every bit of what I was hoping for.

It is quite hefty now too.

I got the idea for the silicone mold from this video on YouTube: How to Make a Tofu 60 Silicone Dampener | Serendipity

Cable:

CableMod Pro Coiled Keyboard Cable (Midnight Black) | $29.90

Tachyon mode seems to be working fine with this cable. I run my keyboard brightness pretty low at 8% though. You could probably run it a lot higher, I just prefer to run it low, so I can see the white pcb plate.

Springs:

I did a full spring swap with 50cN Wooting Springs | $14.99 (*Discontinued*) -replaced by 45cN switch | $39.99

The 50cN springs feel quite a bit lighter than the stock 60cN springs and were definitely worth the time it took to put them in.

The 50cN springs have been replaced by 45cN switches (instead of loose springs) but according to Wooting they are very similar. Nonetheless, I would still love to try them out sometime to see the difference between the two.

For my fellow Counter-Strike players: Jiggle peeking is absolutely insane with 50cN springs inside the Lekker switches. Takes a little time to get used to them, but I can’t go back to normal mechanical switches now, these switches are too good.

If you would still like to purchase loose springs, to swap them out yourself, you might have some luck finding 50cN non-magnetic switch springs, on AliExpress.

Lekker Spring Dimensions:

  • Inner diameter 5.5 mm
  • Wire diameter 0.25~0.3 mm
  • Free length (not pressed) 16.5 mm *(length might change when the start & end gf is different)*
  •  
  • - From Wooting Support

The image on the right shows the difference in spring length between the 60cN spring (left) and 50cN (right) while inside the lekker switch.

Switch Lubing:

$34.99 | when purchased together Wooting Lubricant Set | Wooting Tool Kit

I went with both the Switch Lube Set and Toolkit from Wooting. Both worked great, no complaints.

Not exactly sure how long it took but I have a good estimate... My phone died right after I started, so I pulled out some vinyl records to listen to. Made it through a little over 4-5 albums before I was done, so you can probably estimate the time.

Case:

GH60 Aluminum Keyboard Case (Black) | $79.00

Really happy with this case for $80 and it adds a lot of weight to the keyboard.

If you plan on burger mounting with this case, get something like these...

Screws:

500pcs M2 M2.5 M3 Laptop Notebook Computer Replacement Screws | $6.99

Ordered this screw kit because the stock GH60 case screws are a little too short for my liking and I struggled to get them to thread at all with 2 o-rings.

I used m2x8mm screws but it really depends on the pcb height you want and how much sound damping material you use . (I would suggest using m2x6mm with the stock 60HE foam, but it’s personal preference)

O-Rings:

O-Rings | $7.99

(4mm OD 1mm ID 1.5mm) - Suggested By BadSeed Tech (others sizes work as well)

Keycaps:

PBTfans WOB Crosshair Alphas w/ Japanese Sub-legends (Addon) | $49.00 | & YMDK WOB Katakana Keycaps | $32.90

Did a lot of research to find these PBTFans Crosshair Keycaps but they are only available as an add-on kit, so you need extra keys!

To fill out the rest of the caps I went with a set from YMDK.

Note these keycaps are two different shades of black but I think it adds a unique color scheme to this build and really like how it turned out. It is something to consider if you want to purchase these two keycap sets for yourself though.
